#  Skillink

Skillink is a **Flutter-based service marketplace application** designed to connect skilled professionals with clients who need task-based services. The platform simplifies hiring and service delivery through a modern, efficient, and user-friendly digital system.

The project is currently under active development, with continuous improvements and new features being added. A web-based version is also planned for future release.

---

##  Features

- Secure user authentication system *(in development)*
- Skill-based service listing and discovery
- Client–worker matching system
- Service request management system
- Real-time chat functionality *(planned)*
- Admin dashboard for management *(web version planned)*

---

## 🛠 Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** Firebase / REST APIs *(planned & in progress)*
- **Platforms:** Android, iOS (Web version planned)

##  Project Objective

The goal of Skillink is to bridge the gap between skilled workers and clients by providing a reliable digital platform for service-based work, ensuring convenience, transparency, and efficiency.
